As a Doctor, I want to update my profile with specialization and contact information so that patients have accurate and up‑to‑date details when booking appointments.

Acceptance Criteria
Profile Update Form  
The portal must provide a form where doctors can edit specialization, contact information, and other relevant details.

Field Validation  
The system must validate required fields (e.g., specialization, phone number, email) before saving updates.

Real-Time Update  
Updated information must reflect immediately in the patient-facing directory and booking system.

Confirmation Message  
After successful update, the system must display a confirmation message and log the event.

Error Handling  
If updates fail (e.g., invalid email format), the system must prevent submission and show clear error messages.

Audit Logging  
All profile updates must be logged with doctor ID, timestamp, and updated fields for compliance.

Priority
High — Ensures patients always have reliable information.

Story Points
4–6 points (depending on complexity of validation and synchronization with patient views).
