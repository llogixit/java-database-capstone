As a Doctor, I want to view my appointment calendar in the portal so that I can stay organized and manage patient consultations effectively.

Acceptance Criteria
Calendar Dashboard  
The portal must provide a calendar view where doctors can see all scheduled appointments.

Daily/Weekly/Monthly Views  
Doctors must be able to toggle between daily, weekly, and monthly views for better planning.

Appointment Details  
Each appointment entry must display patient name, time, duration, and consultation type.

Filter and Search  
Doctors must be able to filter appointments by patient, date, or type of consultation.

Real-Time Updates  
Any changes (new bookings, cancellations, reschedules) must update the calendar in real time.

Secure Access  
Only logged-in doctors can view their own calendar; no cross-access between accounts.

Priority
High — Essential for doctors to manage their schedules and patient care.

Story Points
6–8 points (depending on complexity of calendar views, filters, and real-time updates).
