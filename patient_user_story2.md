As a Patient, I want to sign up using my email and password so that I can securely book appointments with doctors through the portal.

Acceptance Criteria
Registration Form  
The portal must provide a sign-up form where patients can enter their email, password, and basic profile information (e.g., name, age, contact number).

Email Validation  
The system must validate the email format and check for duplicates to prevent multiple accounts with the same email.

Password Security  
Passwords must meet security requirements (minimum length, mix of characters) and be stored securely using hashing and salting.

Confirmation Message  
After successful registration, the system must display a confirmation message and send a verification email.

Login Access  
Once registered, patients must be able to log in immediately with their email and password to access appointment booking features.

Error Handling  
If registration fails (e.g., weak password, duplicate email), the system must display clear error messages.

Priority
High — Essential for patient onboarding and appointment booking.

Story Points
5–7 points (depending on complexity of email verification and password security).
